How to Integrate AI Chatbot into React

Step 1: Copy Your Widget Code

Go to your dashboard and copy the integration code snippet.

Step 2: Add to Your App

In your main App.js or index.js file, add the script using useEffect:

useEffect(() => {
const script = document.createElement('script');
script.innerHTML = `/* Your code snippet */`;
document.body.appendChild(script);
}, []);

Step 3: Run Your App

Start your React app and the chatbot will appear.
