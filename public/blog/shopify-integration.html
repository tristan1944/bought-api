How to Integrate AI Chatbot into Shopify

Step 1: Copy Your Widget Code

Go to your dashboard and copy the integration code snippet.

Step 2: Access Theme Editor

In your Shopify admin, go to Online Store → Themes → Actions → Edit code.

Step 3: Edit theme.liquid

Open the theme.liquid file and find the closing </body> tag. Paste your code snippet just before it.

<!-- Paste your chatbot code here --> <script> // Your code snippet </script> </body>

Step 4: Save and Publish

Click "Save" and your chatbot will appear on all pages of your Shopify store.
