How to Integrate AI Chatbot into Squarespace

Step 1: Copy Your Widget Code

Go to your dashboard and copy the integration code snippet.

Step 2: Add Code Injection

In Squarespace, go to Settings → Advanced → Code Injection and paste your code in the "Footer" section.

Step 3: Save

Click "Save" and your chatbot will appear on your Squarespace site.
