How to Integrate AI Chatbot into WordPress

Step 1: Copy Your Widget Code

Go to your dashboard and copy the integration code snippet.

Step 2: Install Code Snippets Plugin (Optional)

For easier management, install the "Code Snippets" plugin from the WordPress repository.

Step 3: Add Code to Your Theme

Go to Appearance → Theme Editor → Footer (footer.php) and paste your code snippet just before the closing </body> tag.

<!-- Paste your chatbot code here --> <script> // Your code snippet </script>

Step 4: Alternative Method - Using a Plugin

Alternatively, you can use the "Insert Headers and Footers" plugin:

  1. Install "Insert Headers and Footers" plugin
  2. Go to Settings → Insert Headers and Footers
  3. Paste your code in the "Scripts in Footer" section
  4. Click "Save"

Step 5: Verify Installation

Visit your WordPress site and check the bottom-right corner. You should see your chatbot widget!
